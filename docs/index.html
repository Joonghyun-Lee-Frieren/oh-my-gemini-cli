Extensions-first release v0.2.0

oh-my-gemini-cli

Context-engineering-powered multi-agent workflow pack for Gemini CLI.

gemini extensions install https://github.com/Joonghyun-Lee-Frieren/oh-my-gemini-cli

"Claude Code's core competitiveness isn't the Opus or Sonnet engine. It's Claude Code itself. Surprisingly, Gemini works well too when attached to Claude Code."

Jeongkyu Shin (CEO of Lablup Inc.), from a YouTube interview

This project started from that observation: "What if we bring that harness model to Gemini CLI?"

Core Model

Native Extension

Implemented with official Gemini CLI extension primitives, centered on gemini-extension.json.

Role-based Agents

Architect, planner, executor, reviewer, debugger, researcher, and quick-fix roles.

Reusable Skills

Plan, execute, team, research, and context optimization workflows.

Namespaced Commands

Consistent /omg:* command set for status, team flow, optimization, and cache checks.

Quick Start

01 INSTALL (TERMINAL)
gemini extensions install https://github.com/Joonghyun-Lee-Frieren/oh-my-gemini-cli
02 VERIFY
gemini extensions list

# or inside interactive mode
/extensions list
03 COMMANDS
/omg:status
/omg:team Implement OAuth auth
/omg:optimize
04 SKILLS
$plan Build migration plan
$execute Implement phase 1
$team End-to-end feature delivery
